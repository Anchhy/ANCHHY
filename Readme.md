# SmartChoice Mobile Store

SmartChoice Mobile Store is a web-based platform for browsing, comparing, and purchasing new and used smartphones from all major brands in Cambodia. The site features a modern, responsive design and a user-friendly interface for seamless shopping.

## Features

- Browse phones by brand (Samsung, Apple, Google, Oppo, Vivo, OnePlus, Xiaomi, Huawei, Asus, Sony, Realme, Honor, Infinix, Tecno, and more)
- Product listings with images, specifications, and prices
- Add products to cart and manage your cart
- Modal product details for more information
- Responsive navigation and mobile-friendly layout
- Contact information and shop location
- Social media links and company information

## Usage

1. Open `index.html` in your browser to start browsing products.
2. Use the navigation bar to explore brands, products, and your cart.
3. Add products to your cart and proceed to checkout.

## File Structure

- `index.html` - Main landing page
- `brand-*.html` - Brand-specific product pages
- `more-products.html` - All products listing
- `cart.html` - Shopping cart page
- `about.html` - About the shop
- `styles.css` - Main stylesheet
- `menu.js`, `script.js` - JavaScript for interactivity

## Customization

- Add or update products by editing the relevant HTML files.
- Update contact or social information in the footer sections.
